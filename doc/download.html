Downloading TestNG

You can download TestNG here.

For the Eclipse plug-in, we suggest using the update site:

  • Select Help / Software updates / Find and Install.
  • Search for new features to install.
  • New remote site.
  • For Eclipse 3.4 and above, enter http://beust.com/eclipse.
  • For Eclipse 3.3 and below, enter http://beust.com/eclipse1.
  • Make sure the check box next to URL is checked and click Next.
  • Eclipse will then guide you through the process.

However, should this method fail, you can also download the TestNG Eclipse plug-in directly and extract it in your eclipse/ directory.
